# Constants

This folder contains all application constants and configuration values.

## Files

- **chatConstants.ts** - Chat-related constants including quick actions and suggestions

## Usage

Import constants from this folder in your components:

```typescript
import { QUICK_ACTIONS, INTRO_SUGGESTIONS } from "../constants/chatConstants";
```

## Adding New Constants

When adding new constant files:

1. Create a new `.ts` file in this folder
2. Export your constants
3. Update this README with the new file description
